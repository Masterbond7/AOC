Start:
    Free!

Open input file:
    (0-7) File descriptor

Get file status and size:
    (0-7) File descriptor
    (8-151) File status

    (0-7) File descriptor
    (8-15) File size (bytes)

Map some memory and load the file:
    (0-7) File descriptor
    (8-15) File size (bytes)
    (16-23) Pointer to file in memory

Unmap the memory for the input file:
    (0-7) File descriptor
    (8-15) File size (bytes)

Get rid of file size:
    (0-7) File descriptor

Close input file:
    Free again! :D
